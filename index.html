My Coding Experience: The Beginning of a New Journey

I quit my full-time job to pursue a different career in my life and I couldn't be any happier. I think I made the right decision as I continue to push myself forward towards becoming a full-stack developer. The learning process thus far has been a roller coaster ride. However, I find myself enjoying every moment of it. I learn as much as I can each day while I am working on the different challenges provided by Codinghouse.

The Learning Process so Far:

Where do I begin? I ask myself this every time I work on a new challenge. Thoughts keep running through my mind over and over again. As a result, I take notes and write down the steps on how to achieve the challenge. For example, when I am working on my website, I would sketch out the layout of how I want to achieve it and what content to add. After, I would then write down some notes on what code is needed to create the different sections for the website. For the Javascript challenges, I would also do the same and write down what I can use to complete the program and then use JSBIN to help write and debug my code. I find JSBIN to be really helpful.
