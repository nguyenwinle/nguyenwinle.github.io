My Coding Experience: The Beginning of a New Journey

I left my full-time job on June 6, 2016 to pursue a career in web and mobile development and I couldn't be any happier. I think I made the right decision as I learn how to code while I am working on the different coding challenges provided by Codinghouse. The learning process thus far has been a roller coaster ride. However, I find myself enjoying every moment of it.

The Learning Process so Far:

Where do I begin? I ask myself this question every time I work on a new challenge. Thoughts keep running through my mind over and over again. Therefore, I realized that I should take notes and write down different steps and possibilities on how to achieve the challenges. I would also use google as a tool to help me with the different challenges. For example, when I am working on my website, I would sketch the layout of how I want it to look and the contents I want to add. I would browse through Getbootstrap and see what I can do with my website. After, I would then write down some notes on what code is needed to create the different sections for the website. Building my website was a long process and there was a lot of trial and error. As for the other challenges such as the Javascript ones, I would write down what I can use to complete the problem and then use JSBIN to help write and debug my code...
